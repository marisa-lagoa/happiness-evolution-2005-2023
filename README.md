# Como evoluiu a felicidade nos países mais e menos felizes entre 2005 e 2023?
Este repositório contém uma análise exploratória em Python com base no World Happiness Report 2024.  O projeto foi desenvolvido no âmbito da unidade curricular de  Ciência de Dados: Etapas e Visualização de Dados da Universidade de Évora e responde à seguinte questão: **Como evoluiu a felicidade nos países mais e menos felizes entre 2005 e 2023?**

# Conteúdos do Repositório
fc75418_codigo.ipynb — Notebook Jupyter com todo o código da análise

fc75418_relatorio.pdf — Relatório completo do projeto

fc75418_apresentacao.pdf — Apresentação em formato PDF

README.md — Este ficheiro

# Objetivo do Projeto
O objetivo principal foi analisar a evolução da felicidade ao longo de quase duas décadas, comparando:

- Os 10 países mais felizes de cada ano
- Os 10 países menos felizes de cada ano
  
A análise procura responder a questões como:

- Há uma diferença grande entre os dois grupos?
- Essa diferença mantém-se estável ao longo do tempo?
- Quais são os países que aparecem mais vezes no topo e no fundo do ranking?
- Existem padrões de estabilidade ou volatilidade?
