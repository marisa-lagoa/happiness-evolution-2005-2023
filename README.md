# happiness-evolution-2005-2023
Este repositório contém uma análise exploratória em Python sobre a evolução da felicidade nos países mais e menos felizes do mundo entre 2005 e 2023, com base no World Happiness Report 2024.  O projeto foi desenvolvido no âmbito da unidade curricular de  Ciência de Dados: Etapas e Visualização de Dados da Universidade de Évora
